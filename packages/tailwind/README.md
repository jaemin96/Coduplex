## 📖 개요

`React + TypeScript + Vite` 기반으로 구축되어 여기에 `SCSS`가 아닌 <br />
`Tailwind css` 스타일도구도 사용해서 개발해보면서 여러 방면으로 역량 강화에 도움을 주기 위한 패키지

<br/>

## 🧫 방향성

- `Tailwind css` 실습을 통한 역량 강화

- 구축 속도가 중요한 경우 가볍고 빠르게 작업할 수 있는 능력 향상

- 규모가 비교적 작은 서비스의 경우 하나부터 열까지 CSS 작업을 진행하지 않고 사용할 수 있게 숙련도 높이기

<br />

## ⚙️ 개발 환경 및 품질 관리

- Tailwind를 사용해보면서 느꼈던 classname 지옥을 별도의 classes 파일로 빼서 관리

- classnames 라이브러리와 병합해서 분리한 classes들을 유연하게 병합

## 🤝 협업 및 코드 표준화

- SCSS 7-1 Layer를 주로 사용해왔는데 이런 설정들을 Tailwind에서도 동일하게 설정할 수 있어서 mixin, variables 등 유사한 설정들을 Tailwind에서도 적용해볼 예정
